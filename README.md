# rail_report

## Ideer
* Busstrafik i Uppsala, UL
* KoDa API
* Drill down för att välja busslinje/hållplats, samt vecka/månad/tid
* Varje busslinje och hållplats:
  * Antal avgångar
  * Andel avgångar i tid
  * Andel avgångar försenade
  * Snitt på försenade avgångar
  * Utveckling av förseningar för linje/hållplats över tid? (diagram)
  * Andel förseningar per tidsintervall på dygnet
  * Automatiskt genererade insikter av rapporten
  * Spontan tanke, heat map över staden som visar vart mest förseningar skett under vald period?
